### Install Node.js

### Install three.js :
```bash
npm install --save three
```

### Install vite :
```bash
npm install --save-dev vite
```

### Clone Repo :
```bash
git clone https://github.com/AmadNaseem/three_js_project.git
```
### Install Dependencies :
```bash
cd three_js_project

npm install
```

### Run code :
```bash
npx vite
```
